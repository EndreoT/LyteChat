<h2>About </h2>
<p> Real time secure web chat application. Includes Blazor web client and REST API. </p>

<h2>Technologies</h2>
<ul>
  <li> C# </li>
  <li> ASP.NET Core 5 </li>
  <li> MySQL </li>
  <li> Entity Framework</li>
  <li> SignalR </li>
  <li> Protected API and chat hub authentication using JSON Web Tokens </li>
  <li> Blazor </li>
  <li> Bootstrap </li>

 </ul>
